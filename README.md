# One Night Werewolf Online

## [Play the game](https://lerman01.github.io/OneNightUltimateWerewolf/)

Serverless version of One Night Werewolf which based one WebRTC protocol 
to communicate between the players.

- #### Vampires expansion
- #### Roles:
